Pacman

This project is a simple console-based version of the classic arcade game Pac-Man, developed using the C programming language. The main objective of the game is to control Pac-Man as he moves through a maze, collecting pellets while avoiding a moving ghost. The maze is represented using ASCII characters, making it entirely playable within a terminal window. This project demonstrates important programming concepts such as arrays, loops, conditionals, user input handling, and randomization. It serves as an engaging example of how classic game mechanics can be recreated with fundamental C programming techniques.
